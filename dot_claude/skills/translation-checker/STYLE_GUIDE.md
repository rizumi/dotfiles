## 日本語スタイルガイド

### カテゴリ別の文体

| カテゴリ | 文末表現 | 例 | 説明 |
|---------|---------|-----|------|
| ボタンラベル | 名詞形・動詞形 | 完了、キャンセル、選択 | 簡潔に |
| 画面タイトル | 名詞形 | アプリ情報、設定 | シンプルに |
| 指示メッセージ | 丁寧な命令形 | ～してください | 敬語を使用 |
| 状態メッセージ | です・ます調 | ～です、～しました | 客観的に |
| エラーメッセージ | です・ます調 | ～できません、～に失敗しました | 丁寧に |
| 確認ダイアログ | 疑問形 | ～しますか？ | 丁寧に確認 |
| 説明文（短文） | です・ます調 | ～です、～します | 明確に |
| 説明文（長文） | です・ます調 | ～してください、～になります | 詳しく |
| プライバシー記述 | です・ます調 | ～に利用します | 目的を明確に |

### 敬語レベル

#### 標準レベル（丁寧語）
ユーザーへの指示や説明

```
推奨:
- 「選択してください」
- 「保存します」
- 「完了しました」

避ける:
- 「選択しろ」（命令形）
- 「選択して」（普通形）
```

#### 高敬語レベル
エラーやトラブル発生時

```
推奨:
- 「お手数ですが、もう一度お試しください」
- 「申し訳ございません」

避ける:
- 「もう一度やって」（カジュアルすぎる）
```

#### 非敬語レベル
純粋な状態表示やタイトル

```
推奨:
- 「設定」
- 「アプリ情報」
- 「選択中」

避ける:
- 「設定します」（不要な敬語）
```

### ボタンラベル

#### 名詞形を優先
```
✓ 完了
✓ キャンセル
✓ 選択
✓ 削除

✗ 完了する
✗ キャンセルする
✗ 選択してください（冗長）
```

#### 例外: アクションが明確でない場合
```
✓ 詳細を見る
✓ 設定を開く
```

### メッセージ

#### 指示メッセージ
```
✓ 写真を選択してください
✓ アルバムを指定してください

✗ 写真を選択して（カジュアルすぎる）
✗ 写真選択（簡潔すぎて不親切）
```

#### 状態メッセージ
```
✓ 保存しました
✓ 削除に成功しました
✓ 同期中です

✗ 保存した（カジュアルすぎる）
✗ 保存完了（簡潔すぎる）
```

#### エラーメッセージ
```
✓ ファイルを保存できませんでした
✓ ネットワークエラーが発生しました
✓ お手数ですが、もう一度お試しください

✗ エラー（不親切）
✗ 保存失敗（冷たい）
✗ 保存できなかった（カジュアルすぎる）
```

### 確認ダイアログ

#### 疑問形で丁寧に
```
✓ 削除してもよろしいですか？
✓ 保存しますか？
✓ 変更を破棄しますか？

✗ 削除する？（カジュアルすぎる）
✗ 本当に削除？（不完全な文）
```

### 数字と単位

#### スペースなし
```
✓ 5秒
✓ 10個
✓ 3.5MB

✗ 5 秒（スペースあり）
✗ 10 個
```

#### 助数詞の統一
```
写真: 枚
ファイル: 個
秒数: 秒
```

### 句読点

#### 句点（。）の使用
```
文末には句点をつける:
✓ 「ファイルを保存しました。」

例外（短いメッセージ）:
✓ 「完了」
✓ 「保存しました」（短文は省略可）
```

#### 疑問符（？）の使用
```
確認ダイアログ:
✓ 「削除しますか？」

例外なし、必ず付ける
```

## 韓国語スタイルガイド

### 존댓말 (丁寧語) の使用

#### 기본 원칙
UI 텍스트는 존댓말 사용

```
✓ 선택해 주세요 (選択してください)
✓ 저장했습니다 (保存しました)
✓ 삭제하시겠습니까? (削除しますか？)

✗ 선택해 (非丁寧語)
✗ 저장했어 (非丁寧語)
```

### 격식체 vs 비격식체

#### 격식체 (格式体) - フォーマル
```
✓ 선택하십시오 (お選びください)
✓ 저장하셨습니다 (お保存になりました)

使用場面: 公式アプリ、ビジネスアプリ
```

#### 비격식체 (非格式体) - カジュアル
```
✓ 선택하세요 (選択してください)
✓ 저장했어요 (保存しました)

使用場面: 一般アプリ、カジュアルアプリ
```

**推奨: 비격식체（カジュアル）を使用**
- 親しみやすさ
- 現代的な印象

### 카테고리별 문체

| 카테고리 | 문체 | 예시 |
|---------|------|------|
| 버튼 라벨 | 명사형/동사형 | 완료, 취소, 선택 |
| 화면 제목 | 명사형 | 앱 정보, 설정 |
| 지시 메시지 | 존댓말 명령형 | ~해 주세요 |
| 상태 메시지 | 존댓말 평서형 | ~입니다, ~했습니다 |
| 오류 메시지 | 존댓말 평서형 | ~할 수 없습니다 |
| 확인 대화상자 | 존댓말 의문형 | ~하시겠습니까? |

### 버튼 라벨

```
✓ 완료 (完了)
✓ 취소 (取消)
✓ 선택 (選択)
✓ 삭제 (削除)

✗ 완료하기 (冗長)
✗ 취소해 주세요 (冗長)
```

### 메시지

#### 지시 메시지
```
✓ 사진을 선택해 주세요
✓ 앨범을 지정해 주세요

✗ 사진 선택해 (非丁寧語)
```

#### 오류 메시지
```
✓ 파일을 저장할 수 없습니다
✓ 네트워크 오류가 발생했습니다

✗ 오류 (不親切)
✗ 저장 실패 (冷たい)
```

### 띄어쓰기

```
✓ 사진 선택 (適切な띄어쓰기)
✓ 앱 정보 (適切な띄어쓰기)

✗ 사진선택 (띄어쓰기 없し)
```

### 숫자와 단위

#### 띄어쓰기 없음
```
✓ 5초
✓ 10개
✓ 3.5MB

✗ 5 초
✗ 10 개
```

## 英語スタイルガイド

### Tone

#### Formal vs. Casual

**Formal:**
```
✓ Please select a photo
✓ Would you like to delete this file?
✓ An error has occurred

使用場面: Enterprise apps, Professional tools
```

**Casual:**
```
✓ Pick a photo
✓ Delete this file?
✓ Oops! Something went wrong

使用場面: Consumer apps, Social apps
```

**推奨: Friendly & Professional (中間)**
```
✓ Select a photo
✓ Delete this file?
✓ Something went wrong
```

### Category-Based Style

| Category | Style | Example |
|---------|-------|---------|
| Button Label | Verb/Noun | Done, Cancel, Select |
| Screen Title | Noun | Settings, App Info |
| Instruction | Imperative/Request | Select a photo, Please try again |
| Status Message | Statement | Saved, Deleted |
| Error Message | Polite explanation | Cannot save file |
| Confirmation | Question | Delete this file? |

### Button Labels

#### Verb form preferred
```
✓ Done
✓ Cancel
✓ Select
✓ Delete

✗ To Do (不自然)
✗ Canceling (進行形は避ける)
```

#### Capitalize first letter only
```
✓ Delete all
✓ Select photo

✗ Delete All (Title Case - Apple HIGでは非推奨)
✗ SELECT PHOTO (全大文字)
```

### Messages

#### Instructions
```
✓ Select a photo from your album
✓ Tap to choose

Casual:
✓ Pick a photo

Formal:
✓ Please select a photo from your album
```

#### Status Messages
```
✓ Saved
✓ Photo deleted
✓ Syncing...

✗ Save completed (冗長)
✗ The photo has been deleted (冗長)
```

#### Error Messages
```
✓ Cannot save file
✓ Network error occurred
✓ Please try again

✗ Error (不親切)
✗ Save failed (冷たい)
```

### Confirmation Dialogs

```
✓ Delete this photo?
✓ Are you sure?
✓ Discard changes?

✗ Do you want to delete? (冗長)
✗ Really delete? (カジュアルすぎる可能性)
```

### Numbers and Units

#### Space between number and unit
```
✓ 5 s
✓ 10 items
✓ 3.5 MB

✗ 5s (スペースなし)
✗ 10items
```

### Punctuation

#### Period
```
Short messages: No period
✓ "Saved"
✓ "Photo deleted"

Long messages: Use period
✓ "Your changes have been saved."
✓ "The photo has been deleted from your album."
```

#### Question mark
```
Always use for questions:
✓ "Delete this photo?"
✓ "Are you sure?"
```

## 中国語スタイルガイド

### 语气选择

#### 正式 vs. 随意

**正式:**
```
✓ 请选择照片
✓ 您确定要删除此文件吗？

使用场景: 企业应用、专业工具
```

**随意:**
```
✓ 选择照片
✓ 删除此文件？

使用场景: 消费者应用、社交应用
```

**推荐: 友好且专业（中等正式度）**

### 分类文体

| 分类 | 文体 | 示例 |
|------|------|------|
| 按钮标签 | 动词/名词 | 完成、取消、选择 |
| 屏幕标题 | 名词 | 设置、应用信息 |
| 指示消息 | 祈使句 | 请选择照片 |
| 状态消息 | 陈述句 | 已保存、已删除 |
| 错误消息 | 礼貌说明 | 无法保存文件 |
| 确认对话框 | 疑问句 | 是否删除此文件？ |

### 按钮标签

```
✓ 完成
✓ 取消
✓ 选择
✓ 删除

✗ 完成操作 (冗余)
✗ 请取消 (冗余)
```

### 消息

#### 指示消息
```
✓ 请选择照片
✓ 请指定相册

可选: "请"可省略（根据应用调性）
✓ 选择照片
```

#### 错误消息
```
✓ 无法保存文件
✓ 发生网络错误

✗ 错误 (不友好)
✗ 保存失败 (生硬)
```

### 标点符号

#### 句号
```
短消息: 可省略
✓ 已保存
✓ 已删除照片

长消息: 使用句号
✓ 您的更改已保存。
```

#### 问号
```
疑问句必须使用:
✓ 是否删除此照片？
✓ 确定吗？
```

### 数字与单位

#### 无空格
```
✓ 5秒
✓ 10个
✓ 3.5MB

✗ 5 秒
✗ 10 个
```

## 言語横断的なスタイル原則

### 1. 一貫性
同じカテゴリのテキストは同じトーンを維持

### 2. 簡潔性
特にボタンラベルやタイトルは簡潔に

### 3. 親しみやすさ
ユーザーフレンドリーだが、過度にカジュアルではない

### 4. 明確性
曖昧さを避け、ユーザーが理解しやすい表現

### 5. 敬意
エラー時は特に丁寧に

## プロジェクト固有のスタイルガイド

**重要: プロジェクトに`TRANSLATION_GUIDELINES.md`が存在する場合、それを優先**

プロジェクトのドキュメントから以下を確認：
- トーンの方針（フォーマル/カジュアル）
- ターゲットユーザー層
- ブランドボイス
- 既存の翻訳パターン

## スタイルチェックリスト

翻訳後、以下を確認：

- [ ] 文体がカテゴリに適しているか
- [ ] 敬語レベルが統一されているか
- [ ] トーンが一貫しているか
- [ ] 句読点が適切に使用されているか
- [ ] 数字と単位の表記が言語慣習に合っているか
- [ ] ボタンラベルが簡潔か
- [ ] エラーメッセージが丁寧か
- [ ] プロジェクト固有のスタイルガイドに準拠しているか

|------|------|--------|------|
| Data | データ | データー | 末尾の長音は省略しない |
| Server | サーバー | サーバ | 末尾の長音は省略しない |
| User | ユーザー | ユーザ | 末尾の長音は省略しない |
| Computer | コンピューター | コンピュータ | 末尾の長音は省略しない |
| Printer | プリンター | プリンタ | 末尾の長音は省略しない |
| Error | エラー | エラ | 末尾の長音は省略しない |
| Browser | ブラウザー | ブラウザ | 末尾の長音は省略しない |

## 言語別の注意点

### 日本語

#### 文字種の使い分け
```
漢字: 専門用語、動作（選択、削除、保存）
ひらがな: 助詞、補助動詞（～してください）
カタカナ: 外来語、技術用語（アプリ、ファイル）
英数字: ブランド名、固有名詞（Face ID, iOS）
```

#### 数字の表記
```
推奨: アラビア数字（1個、10枚）
避ける: 漢数字との混在（一個、10枚）
```

### 韓国語

#### 한글과 외래어
```
고유어 우선: 선택 (選択), 삭제 (削除)
외래어 사용: 앨범 (album), 슬라이드쇼 (slideshow)
```

#### 띄어쓰기
```
正: 앱 스토어 (App Store - 固有名詞はそのまま)
正: 사진 선택 (写真選択 - 適切な띄어쓰기)
誤: 사진선택 (띄어쓰기 없し)
```

### 中国語（簡体字）

#### 简化字使用
```
推奨: 设置 (設置)
避ける: 設置 (繁体字との混在)
```

#### 专有名词
```
保持原文: Face ID, App Store
本地化: 隔空投送 (AirDrop), 隔空播放 (AirPlay)
```
